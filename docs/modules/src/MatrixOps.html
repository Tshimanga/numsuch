proc wFromPG(con: Connection, edgeTable: string, fromField: string, toField: string, wField: string, n: int, weights = true)¶
Arguments:
  • con – A CDO Connection to Postgres
  • edgeTable – The table in PG of edges
  • fromField – The field of edgeTable containing the id of the head vertex
  • toField – the field of edgeTable containing the id of the tail vertex
  • wField – The field of edgeTable containing the weight of the edge
  • n – number of distinct vertices. In practice, this may be gives and the number of names
  • weights – Boolean on whether to use the weights in the table or a 1 (indicator)
